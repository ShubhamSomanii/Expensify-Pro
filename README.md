# Expensify Pro 💰

Expensify Pro is a modern expense tracking web application built with **React** that helps users manage their spending, track monthly budgets, and analyze financial activity in a clean and interactive interface.

The application allows users to record expenses, categorize transactions, and view their spending insights while also supporting real-time currency conversion using an external API.

---

## 🚀 Features

- Add, edit, and manage expenses
- Categorize transactions for better tracking
- Monthly budgeting and spending overview
- Real-time currency conversion using ExchangeRate API
- Responsive and modern UI
- Smooth UI interactions and animations
- Export expense data (CSV support)
